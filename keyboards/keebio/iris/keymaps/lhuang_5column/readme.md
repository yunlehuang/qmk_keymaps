## Layers

### _QWERTY
Base layer using QWERTY. Can access the `_SYM`, `_NAV`, `_NUM`, and `_MUS` layers from here. Another thing to note is that `shift` is on the left thumb.

### _SYM
Layer dedicated to symbols.

### _NAV
Navigation keys on the right half of the keyboard. Use of `ctrl` + arrow keys allows easy navigation between words. The keys on the left allows for easy navigation between chrome keys.

### _NUM
Numberpad and F keys.

### _GAM
Gaming layer. The main change is that there are no layer keys on the left half. This way, all thumb buttons can be fully utilized. The right thumb button goes back to QWERTY.

### _MOU
Controls the mouse.

### _MUS
Controls music/media settings. Also takes you to the _GAM layer and the _MAC layer (swaps the `GUI` and `CTRL` keys). It also allows you to access the `reset` key.
